# 🧠 AI Decision Simulator

An intelligent MERN stack web application that helps users make smarter decisions using structured comparison, weighted scoring, and AI-inspired evaluation techniques.


# 📌 Overview

AI Decision Simulator is a full-stack web application designed to simplify complex decision-making.

Users can:

* Compare multiple options
* Assign priorities and weights
* Analyze scores visually
* Track decision history
* Manage profiles securely

The platform provides a clean and interactive interface with real-time analysis to help users make more confident decisions.
